Приложение для складского учета

  • Для добавления товаров перейдите в Товары
  • Создайте закупку, добавьте в него все расходы связанные с этой закупкой
  • Установите статус "Оплачен", чтобы автоматически добавить дебет, затем установите статус "Доставлен", чтобы можно было пополнить склад
  • Создайте склад, теперь можно пополнить его закупками
  • Создайте покупателя, теперь можно создать заказ
  • Перейдите к списку покупателей, выберите покупателя и создайте заказ, добавьте в него продукты со склада
  • Переведите статус заказа в "Оплачено" чтобы автоматически создать кредит
  • Переведите статус заказа в "Доставлен" чтобы автоматически списать товары со склада
  • Добавляйте кредиты или дебиты по мере надобности
  • Перейдите в баланс, чтобы посмотреть баланс за период
